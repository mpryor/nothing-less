# nless

Alias package for [nothing-less](https://pypi.org/project/nothing-less/) — a TUI pager for exploring tabular data with vi-like keybindings.

```bash
pip install nless
# or
pipx install nless
```

This installs the `nless` command. See the [nothing-less repo](https://github.com/mpryor/nothing-less) for full documentation.
